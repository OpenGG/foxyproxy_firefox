Зразки URL-адрес

Перед тим, як Firefox завантажить URL-адреси, Firefox питає FoxyProxy який проксі потрібно використовувати для цього. FoxyProxy відповідає на це питання порівнюючи дану URL-адресу зі всіма зразками URL-адрес, які ви визначите. Найпростіший шлях вказання зразків за допомогою метатегів.

Метатеги

Метатеги обчислюються глобально; ви маєте завчасно визначити їх перед наперед. Зірочка (*) заміни, як знак підстановки знаку для нуля або інших знаків, і знак питання (?) заміни, як знак підстановки знаку для будь-якого знаку.

Більша відповідність правилам можлива при використанні регулярних виразів. Для отримання більш конкретної інформації, клацніть кнопку "Зміст Допомоги".
