URL образцы

Перед загрузкой страницы (URL) Firefox выясняет у FoxyProxy должен ли использоваться прокси. FoxyProxy отвечает на запрос попыткой найти соответствие между текущим URL и теми URL-шаблонами, которые определены ниже. Самый простой способ создать шаблон - использовать метасимволы (wildcards).

Метасимволы

Используя регулярные выражения можно составить более сложные правила. Нужны подробности - нажмите кнопку "Помощь".

Примеры использования метасимволов

URL шаблонАдрес соответствует шаблонуАдрес не соответствует шаблону

http://mail.yahoo.com/inbox/123.html

http://digg.com/http://digg.com/http://digg.com
http://www.digg.com/
