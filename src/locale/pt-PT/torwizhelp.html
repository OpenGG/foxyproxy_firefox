Padrões de Endereço

Antes do Firefox carregar uma página, o Firefox pergunta ao FoxyProxy se um proxy deve ser usado. O FoxyProxy responde esta questão procurando por algum padrão de endereço. A maneira mais fácil de se definir padrões de endereço é usando caracteres curingas.

Os asteriscos (*)

Regras mais avançadas para padrões são possíveis usando expressões regulares. Para detalhes, clique no botão "Conteúdo de Ajuda".
